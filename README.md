# WIP

This layout aims to make the transition from Autodesk Inventor to FreeCAD as smooth as possible. It mimics the user interface of Autodesk Inventor, including the placement of tools and commands, to help users feel more comfortable and familiar with FreeCAD.

The layout is consumed by the fantastic addon [APEbbers/FreeCAD-Ribbon: A customizable ribbon UI for FreeCAD.](https://github.com/APEbbers/FreeCAD-Ribbon)

## Installation

### FreeCAD-Ribbon

#### Install via the Addon Manager

1. Open FreeCAD [Addon Manager](https://wiki.freecad.org/Std_AddonMgr).
1. Search for 'FreeCAD-Ribbon'.
1. Click on 'Install'.

Result: Addon will be installed and FreeCAD will prompt for a restart.
Accept prompt and FreeCAD will restart with the Ribbon addon included.

Note: See [wiki page/installation](https://github.com/APEbbers/FreeCAD-Ribbon/wiki/01-%E2%80%90-Installation) for other ways to install.

### Layout

1. Download the layout JSON file from this repository.
2. Open FreeCAD.
3. Open the FreeCAD-Ribbon settings by pressing `Alt + L` or by navigating to the settings menu.
4. In the settings menu, click on the "Import Layout" button.
5. Select the downloaded JSON file and click "Open".
6. The layout will be imported, you may need to restart FreeCAD for the changes to take effect.
